F1 Updates 🏎️

A simple and stylish website that provides Formula 1 updates, built with HTML and CSS.

This project is designed to showcase race highlights, standings, and news in a clean, user-friendly format.

🚀 Features

🏎️ Responsive design for desktop and mobile
🏎️ Easy navigation with a modern layout
🏎️ Dedicated sections for race updates and driver/team info
🏎️ Lightweight and fast (pure HTML + CSS, no frameworks)

📂 Project Structure

📁 index.html → Main webpage
📁 style.css → Styling for the site

🛠️ Setup & Usage

1️⃣ Clone the repository:

git clone https://github.com/IshanD14/F1-Updates.git


2️⃣ Navigate into the project folder:

cd F1-Updates


3️⃣ Open index.html in your browser to view the site.
